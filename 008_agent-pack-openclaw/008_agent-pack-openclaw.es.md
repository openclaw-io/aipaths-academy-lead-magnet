# Pack de Agente IA para OpenClaw

Tu primer agente de IA listo para instalar. "Juan" es un asistente personal especializado en Notion que gestiona tareas, estructura proyectos y mantiene todo organizado.

## Qué Incluye

- **SOUL.md** — La identidad del agente: quién es, su personalidad, cómo se comunica
- **AGENTS.md** — Las operaciones: qué sabe hacer, sus capacidades, reglas de comportamiento
- **Skill de Notion** — Skill completo con script CLI para crear páginas, databases, buscar contenido y más
- **README.md** — Guía paso a paso para instalarlo

## Por Qué Este Pack

La mayoría de la gente instala OpenClaw y no sabe qué hacer después. Este pack te da un agente funcional en 10 minutos para que entiendas cómo funciona el sistema de archivos `.md`, los skills y los scripts.

## Instalación Rápida

1. **Descargar y descomprimir** el pack
2. **Copiar los archivos** a tu directorio de agente en OpenClaw
3. **Configurar tu API key de Notion** como variable de entorno
4. **Iniciar el agente** y empezar a usarlo

## Qué Vas a Aprender

- Cómo se estructura un agente con SOUL.md y AGENTS.md
- Cómo funcionan los skills y scripts
- Cómo conectar un agente con una API externa (Notion)
- Las bases para crear tus propios agentes personalizados
